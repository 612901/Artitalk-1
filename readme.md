* 实时发布
* 支持markdown语法
* 支持剪切板图片直接上传
* 方便引用

## 参与开发
* fork本项目
* 在source文件夹中新建一个版本文件夹，原则上为当前最新版本加0.01
* 将你的修改版本放在新的版本文件夹中后，pr
